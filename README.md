# Alert Component

Simple Componente de alertas

Se puede personalizar:
- Texto de la alerta
- Texto del botón

Uso

- Copiar la carpeta src al proyecto
- Importal el css y js

```js
const alerta = new MyAlert();
alerta.text = "Mi texto";
alerta.btn_text = "Click";

alerta.create__alert();
```
